# Work in Progress

Det jag gör här är baserat på en projektuppgift i kursen Nätverkssäkerhet via Frans Schartau Handelsinstitut, skriven av Alexander Tofer.

Uppgiften är att bygga upp en testmiljö för ett företagsnätverk med två siter på olika geografiska platser: Stockholm och Göteborg. Varje nätverk ska vara segmenterat i olika zoner för att man ska kunna kontrollera trafikflöden och öka säkerheten. Kommunikation mellan de olika nätverken på respektive plats sker genom en ISP med hjälp av en virtuell tunnel.
Det ska även implementeras säker fjärrstyrning av switchar och routrar och ACLs ska läggas upp för att öka säkerheten.
